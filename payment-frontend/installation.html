Get RgentAI up and running in just 3 simple steps

Quick Installation

3 Simple Commands

# 1. Install the package
devtools::install_github("NathanBresette/Rgent-AI", force = TRUE)
# 2. Load the library
library(rstudioai)
# 3. Launch the AI assistant
ai_addin_viewer()

That's it! Your AI assistant is now built right into RStudio.
